# Expense Tracker
A web app to track personal expenses, built with Flask and SQLite.

## Features
You can add income or expense with a selection of categories for each.
You can view transactions, delete individually or reset the database.
A database is created when the app is launched.

## How to run
```bash
git clone https://github.com/veseling02/expense-tracker-sqlite
cd expense-tracker-sqlite
pip install -r requirements.txt
python app.py
```
Open `http://127.0.0.1:5000` in your browser.

## Tech stack
Built using Python, Flask, SQLite

## Purpose
Built to practise Flask and relational databases
